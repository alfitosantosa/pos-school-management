# Product: SMK Fajar Sentosa App

A school management web application for SMK Fajar Sentosa (a vocational high school in Indonesia). The system manages day-to-day school operations across multiple user roles.

## Core Domains

- **Attendance** — Student and teacher attendance tracking per schedule/class, with recap and filtering by date range
- **Schedules** — Class schedules, special schedules, and teacher schedule management
- **Tahfidz** — Quran memorization (tahfidz) group management and student progress records
- **Payments** — Student payment management integrated with Midtrans payment gateway
- **Violations** — Student violation tracking
- **Academic** — Academic year, grade, major, class, and subject management
- **Users & Roles** — Role-based access (admin, teacher, student) via Better Auth
- **Notifications** — WhatsApp bot notifications via Evolution API (botwa)
- **Announcements** — School-wide announcements

## User Roles

- **Admin** — Full system access
- **Teacher** — Manage attendance, schedules, tahfidz records, violations for their classes
- **Student** — View own schedule, attendance, tahfidz records, payment status

## Language

UI text and labels are primarily in **Bahasa Indonesia**.
